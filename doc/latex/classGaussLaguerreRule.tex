\subsection{Detailed Description}
Class derived from \hyperlink{classQuadratureRule}{Quadrature\-Rule} providing the Gauss-\/\-Laguerre quadrature rule. 

Compute nodes and weights for the {\itshape n\-Nodes\-\_\-}-\/points approximation of\-: \[ \int_{0}^{+\infty} w(x)f(x)~\mathrm{d}x \] where $ w(x) = e^{-x} $.

Detailed Description

Abstract class providing methods to assemble matrices to solve one-dimensional PDEs.

Matrices are held in a sparse format.

Assemble the matrix for an advection-diffusion term.

Build the matrix for the advection-diffusion problem: \( -\nabla\cdot\left(\alpha\cdot\gamma\left(\eta\nabla u - \beta u\right)\right) = f \).

Parameters
[in]alpha: \( \alpha \), an element-wise constant function;
[in]gamma: \( \gamma \), an element-wise linear function;
[in]eta: \( \eta \), an element-wise linear function;
[in]beta: \( \beta \), an element-wise constant function.

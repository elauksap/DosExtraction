Detailed Description

Class derived from QuadratureRule providing the Gauss-Hermite quadrature rule.

Compute nodes and weights for the nNodes_-points approximation of:

\[ \int_{-\infty}^{+\infty} w(x)f(x)~\mathrm{d}x \]

where \( w(x) = e^{-x^2} \).
